How to Play

  • Use Left and Right arrow keys or the on-screen joystick (on touch devices) to switch lanes
  • Collect blocks to change your number value
  • Each block represents a mathematical operation:
    • Blue: Addition (+)
    • Red: Subtraction (-)
    • Yellow: Multiplication (×)
    • Green: Division (÷)
  • Try to match your value to the target number when passing through gates
  • Earn points for each successful match!
